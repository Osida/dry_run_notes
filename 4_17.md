SUID/SGID
- find / -type f -perm /2000 -ls 2>/dev/null
    - Finds files with the SGID (Set Group ID) bit set
    - Impact: When a user executes a file with SGID set, the process runs with the privileges of the group that owns the file, rather than the user's group.
- find / -type f -perm /4000 -ls 2>/dev/null
    - Finds files with the SUID (Set User ID) bit set
    - Impact: When a user executes a file with SUID set, the process runs with the privileges of the file owner (often root)
- find / -type f -perm /6000 -ls 2>/dev/null
    - Finds files that have both SUID and SGID bits set
    - Impact: These files are often used for high-privilege system tasks
